## Platform-Specific Tips

### For Midjourney
- Use descriptive, specific keywords
- Order matters (earlier = more influence)
- Combine multiple styles effectively
- Use artist names for style reference

### For Stable Diffusion
- Be very specific with details
- Use negative prompts extensively
- Technical terms work well
- Multiple quality keywords help

### For DALL-E
- Natural language works well
- Be clear and descriptive
- Less need for quality keywords
- Focus on composition and subject
